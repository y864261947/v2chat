## 💡 技术亮点

### 1. CSS 变量系统
利用现有的 `--chatbox-*` 变量体系，无缝替换品牌色，影响全局组件。

### 2. 渐进增强
新样式通过类名添加，不破坏现有功能，易于回滚。

### 3. 性能优化
- CSS 动画使用 GPU 加速（transform）
- 动画循环无 JavaScript 开销
- 样式继承减少 CSS 体积

### 4. 可维护性
- 语义化类名（`.v2chat-*`）
- 完整文档支持
- 清晰的设计令牌
